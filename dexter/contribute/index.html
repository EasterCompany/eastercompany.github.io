Contributing to the Community

At Easter Company, we believe in the power of open collaboration. Most of the core infrastructure that powers Dexter is open source and available for the community to inspect, use, and improve.

Open Source Repositories

dex-cli

The central command-line interface for the Dexter ecosystem. Manage service lifecycles, builds, and system health.

Source Docs

dex-event-service

Central Nervous System. A Redis-backed asynchronous event bus that decouples services and manages communication.

Source Docs

dex-discord-service

Integrates Discord with the Dexter ecosystem. Acts as both an event producer and consumer for chat interactions.

Source Docs

dex-tts-service

High-quality Text-to-Speech powered by Coqui XTTS-v2. Enables Dexter to speak with voice cloning capabilities.

Source Usage

dex-web-service

Web Intelligence & Rendering. Provides headless browser rendering for link unfurling and content extraction.
