Contributing to the Community

At Easter Company, we believe in the power of open collaboration. Most of the core infrastructure that powers Dexter is open source and available for the community to inspect, use, and improve.

How to get involved:

  • GitHub: Star our repositories, report bugs via issues, or submit Pull Requests for new features and fixes.
  • Discord: Join our community to discuss development, share ideas, and get help from other contributors.
  • Early Access: Active contributors on GitHub are regularly granted early access to new features and invited to our private developer channels on Discord.

Our Services & Repositories

The core components of the Dexter ecosystem.

dex-cli

Unified Management Tool

The central command-line interface for the Dexter ecosystem. Manage service lifecycles, builds, deployments, and system health with a single tool.

dex-event-service

Central Nervous System

A high-performance, Redis-backed asynchronous event bus. It decouples Dexter's services and manages all internal communication flows.

dex-discord-service

Discord Integration Bridge

Integrates Discord with the Dexter ecosystem. Acts as both an event producer (monitoring Discord) and an event consumer (executing actions).

dex-tts-service

Text-to-Speech Engine

High-quality TTS powered by Coqui XTTS-v2. Enables Dexter to speak with zero-shot voice cloning capabilities.

dex-web-service

Web Intelligence & Rendering

Provides headless browser rendering and metadata extraction. Allows Dexter to "see" and understand web content for link unfurling and browsing.
